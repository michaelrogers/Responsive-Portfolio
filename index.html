About me

I have a degree in Information Systems from Texas A&M University. I love all things technology, which includes audio, arduino, robotics, VR, and web technologies.

I currently work as a assistant project manager at a software development company here in Houston and have experience providing QA for both web applications and games.

Since early this year, I have dedicating myself towards personal projects which include using vanilla JavaScript and frameworks like Meteor and React. By joining the bootcamp, I hope gain a deeper understanding of how web technologies interact and gain familiarity with some of the latest tools to make it into the industry.
